Acids

Important Facts:

*Blue highlighted are the more important notes

  • An acid is a substance that produces H+/Hydronium ions when it is in a solution.
  • Formula start with H.
  • Name includes "acid".
  • Strong acids are 100% ionized
  • Weak acids are not

Naming Acids

If it is binary and name ends in -ide, then it changes to hydro ___ -ic acid.

e.g. HCl = hydrogen chloride = hydrochloric acid.


If it is polyatomic, and the name ends in -ite, then it changes to ___ -ous acid.

e.g. H2SO3 = hydrogen sulfite = sulfurous acid.


If it is polyatomic, and the name ends in -ate, then it changes to -ic acid.

e.g. HNO3 = hydrogen nitrate = nitric acid.

Making Formulas

Formulas for acids follow the same rules as ionic or polyatomic molecules, because they are ionic or polyatomic molecules, they just also produce H+ ions.
