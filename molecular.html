Moleculars

Important Facts:

*Blue highlighted are the more important notes

  • Forms between 2+ nonmetals
  • Can be multiple combinations of the same atoms
  • Name ends with -ide
  • If a vowel is at the end, it is dropped if element starts with a vowel
  • If it is mono-, it is omitted if only one of the first atom
